# SEO Checklist

## On-Page SEO

### Content
- [ ] Primary keyword in title tag (front-loaded)
- [ ] Primary keyword in H1
- [ ] Primary keyword in first 100 words
- [ ] Secondary keywords in H2/H3 subheadings
- [ ] Keyword density: 1-2% (natural usage, no stuffing)
- [ ] Content length appropriate for topic (min 300 words)
- [ ] Unique content (no duplication)

### Meta Tags
- [ ] Title tag: 50-60 characters
- [ ] Meta description: 150-160 characters with CTA
- [ ] Open Graph tags (og:title, og:description, og:image)
- [ ] Twitter Card tags

### Structure
- [ ] Single H1 per page
- [ ] Logical heading hierarchy (H1 → H2 → H3)
- [ ] Short paragraphs (2-3 sentences)
- [ ] Bullet points and numbered lists where appropriate
- [ ] Table of contents for long-form content

### Links
- [ ] 3-5 internal links to relevant pages
- [ ] External links to authoritative sources
- [ ] Descriptive anchor text (not "click here")
- [ ] No broken links

### Images
- [ ] Descriptive alt text on all images
- [ ] Compressed file sizes (< 200KB where possible)
- [ ] Descriptive file names (not `IMG_1234.jpg`)
- [ ] Lazy loading for below-the-fold images

## Technical SEO

- [ ] Page loads in < 3 seconds
- [ ] Mobile-responsive design
- [ ] HTTPS enabled
- [ ] Clean URL structure (no query parameters for content pages)
- [ ] XML sitemap submitted
- [ ] robots.txt properly configured
- [ ] Canonical URLs set
- [ ] Structured data (JSON-LD) implemented
- [ ] No duplicate content issues
- [ ] 404 page configured
