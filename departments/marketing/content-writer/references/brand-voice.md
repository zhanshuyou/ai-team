# Brand Voice Guide

## Voice Attributes

| Attribute | We Are | We Are Not |
|-----------|--------|------------|
| Tone | Confident, warm, helpful | Arrogant, cold, salesy |
| Language | Clear, conversational, direct | Jargon-heavy, formal, vague |
| Perspective | Empowering, collaborative | Condescending, preachy |

## Writing Do's and Don'ts

### Do
- Use active voice: "We built this for you" (not "This was built")
- Address the reader directly: "you", "your"
- Use concrete examples over abstract claims
- Keep sentences under 25 words
- Use contractions naturally: "you'll", "we're", "it's"

### Don't
- Use buzzwords: "synergy", "leverage", "disrupt"
- Over-use exclamation marks
- Make unsubstantiated claims: "the best", "revolutionary"
- Use passive voice unless intentional
- Write walls of text without visual breaks

## Headline Formulas

1. **How to [achieve result]**: "How to Ship Features 2x Faster"
2. **Number + Benefit**: "5 Ways to Reduce Code Review Time"
3. **Question**: "Are Your API Endpoints Actually Secure?"
4. **Direct benefit**: "Ship Faster Without Sacrificing Quality"

## Formatting Standards

- Use sentence case for headings (not Title Case)
- One space after periods
- Use em dashes (—) for emphasis, not hyphens
- Oxford comma: "design, build, and ship"
